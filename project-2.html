Holbegram

Holbegram is a Instagram clone showcasing my ability to build mobile applications for both iOS and Android using Dart and Flutter.

Code Link

Project Overview

In this project, I demonstrate my quick learning and adaptability by learning Dart and Flutter, then building a Mobile Application in a high pressure, limited time frame.

Dart, unlike Javascript which I am more familiar with, is a type-safe language. This means that you are required to define all types for variables and functions. This was a big change from Javascript and uncomfortable at first, but as I continued to practice I was able to understand the benefits and even enjoy solving problems using Dart.

Flutter is an open source UI development kit created by Google. The most unique feature of Flutter is you can create a UI once, And deploy directly to iOS, Android, and Web.
